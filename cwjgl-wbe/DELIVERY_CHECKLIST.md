## 📱 功能演示场景

### 场景 1: 首次使用
```
1. 用户访问 /aichat
   → 自动创建会话
   → 显示欢迎屏
   
2. 用户点击建议问题
   → 发送消息到后端
   → 后端获取业务数据
   → 调用 Doubao LLM
   → 返回回复
   → 前端显示消息 + 来源

3. 确认消息被保存
   → 刷新页面
   → 消息历史仍在
```

### 场景 2: 多轮对话
```
1. 用户问: "请介绍一下蔡文姬服务"
   → AI 回复

2. 用户问: "目前主要覆盖哪些城市？"
   → 后端包含之前的消息历史
   → AI 基于对话上下文回复

3. 历史会话保存
   → 切换到其他会话
   → 切换回此会话
   → 之前的对话内容完整显示
```

### 场景 3: 会话管理
```
1. 创建新会话A
   → 对话几条消息
   → 点击侧栏创建新会话B

2. 在B中进行独立对话
   
3. 切换回A
   → A的消息历史完整保存
   → 可继续在A中对话

4. 查看会话列表
   → 所有会话显示在侧栏
   → 点击任一会话快速切换
```
